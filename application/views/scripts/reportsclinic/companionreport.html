Hinweise / Abweichungen von der Vorlage

COMPANION_ID

Diese ID ist nur innerhalb eines Export-Dokuments eindeutig.
Patient A kann im Export für das Jahr 2020 die ID 5 und im Export des Jahres 2021 die ID 7 haben.
Die Companion_ID ist keine Patienten-ID sondern eine ID für den Behandlungszeitraum.
Im klinischen Export sind das genau die klinischen Fälle.
In der SAPV ist jede Neuaufnahme und jeder Ortswechsel eine neue COMPANION_ID.

alter

zur Anonymisierung wird nur das Jahr, nicht tagesgenau berechnet
Beispiel: Hat der Patient am 2.Januar Geburtstag, wird das Alter für den Besuch am 1.1. und auch für den Besuch am 3.1. auf 50 Jahre gesetzt.

muttersprache_deutsch

wird nicht erfasst. Wenn Übersetzer nötig, wird muttersprache_deutsch auf 0 gesetzt

art_uebersetzer

wird nicht erfasst

entlassart

es wird der Titel der in ISPC hinterlegten Entlassart eingesetzt

erstdiagnose

Hier wird das Datum der eingetragen an dem die Diagnose in ISPC eingetragen wurde.

Nicht beurteilbar / kein Eintrag

Wenn keine Daten für eine Problemkategorie vorliegen, z.B. weil bei einem Patienten zu erst ein Formular mit nicht allen Kategorien ausgefüllt wurde, bleibt das undokumentierte Feld in der Tabelle leer.

Nicht beurteilbar wird als -99 eingetragen
